## Lync / Skype for Business Gotchas

- When appending `?sl=1` query string to a meeting invitation URL, it forces anonymous users into the web app. See also [this](https://support.office.com/de-de/article/Teilnehmen-an-einer-Skype-Besprechung-mit-der-Skype-for-Business-Web-App-4828ad18-ed21-422a-a870-94d676d4b72a)
- "UCWA" (Unified Communications Web API) not supported by O365
